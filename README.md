# dpd-crypt

This custom resource type allows you allows md5 encryption.

## Installation

`$ npm install dpd-crypt`

See [Installing Modules](http://docs.deployd.com/docs/using-modules/) for details.

## Example Usage

```

dpd.md5.get({
    string: 'Text to encode'
}, function (results) {
    // ...
});

```
